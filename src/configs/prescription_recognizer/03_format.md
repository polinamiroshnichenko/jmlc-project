## Формат вывода
Выведи результат в формате JSON со следующей структурой:
```json
{
  "items": [
    {
      "text": "Оригинальный текст анализа с изображения",
      "type": "Тип анализа"
    }
  ]
}
```

### Типы анализов:
* `lab_analysis` — лабораторный анализ
* `instrumental_analysis` — инструментальный анализ
* `doctor` — направление к врачу

**Правила и рекомендации**
* Если исследование указано не полностью или с ошибками, всё равно включи его в список
* Если на изображении нет ни одного исследования, верни пустой список - `"items": []`
* Сохраняй оригинальное написание исследований, включая сокращения и аббревиатуры, но если есть опечатка и ты понимаешь, как ее исправить, то обнови `text` на исправленный вариант
