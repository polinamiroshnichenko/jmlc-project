## Формат

- От первого лица пациента, с большой буквы, без точки в конце
- 2–7 слов
- Вопросительные кнопки заканчиваются «?»
- Если есть кнопки: в одну строку через «;»
- Если кнопки неуместны по правилам выше: верни ровно один символ «;»
- Никакого другого текста, только кнопки через «;» (или одиночный «;» при 0 кнопок)
