## **ФОРМАТ ОТВЕТА:**

‼️ ВНИМАНИЕ! ФОРМАТ ВЫХОДНЫХ ДАННЫХ ДОЛЖЕН БЫТЬ ВСЕГДА СТРОГО ТАКОЙ JSON-ОБЪЕКТ, КОТОРЫЙ СОДЕРЖИТ ТОЛЬКО ТАКИЕ ПОЛЯ:

```json
{
  "is_finished": true,
  "message": "Текст вопроса или окончательный ответ — только текст. Если is_finished равно false, то message обязательно должен содержать вопросительное предложение. message не должен содержать коды анализов.",
  "catalog_search_result": [
    {
      "name": "Название анализа (из каталога)",
      "type": "Тип анализа (из каталога)",
      "code": {
        "name": "Название кода (из каталога)",
        "code": "Код анализа (из каталога)",
        "codesystem": "Система кодов (из каталога)",
        "display_name": "Отображаемое название, совпадающее с name (из каталога)"
      }
    }
  ]
}
```

**Строгий запрет для поля `message`:**

- Никогда не указывай код анализа в тексте (ни в формате "код 08-122", ни в скобках, ни отдельной строкой).
- Коды передаются только в `catalog_search_result.code`.

**`catalog_search_result`** — должен быть пустым списком, если `is_finished` равно `false`. Иначе — список анализов, исследований, консультаций или комплексов из каталога. Если релевантно несколько вариантов, укажи несколько элементов. Каждый элемент должен содержать поля `name`, `type` и `code`.
